  • 856. 括号的分数｜难度：中等

解题思路

  • 维护一个栈，初始元素为 0

  • 当遇到 ( 时，将 0 压栈

  • 遇到 ) 时，将栈顶元素 top 弹出，如果为 0，则把此时的栈顶加 1；否则，加上 top * 2

  解释：top 为 0 时，意味着遇到了 ()，计算结果为 1；不为 0 时，代表括号内包含其它内容，那么将其计算结果乘二后加到父级括号上。最终算法时间复杂度 O(n)O(n)O(n)，空间复杂度 O(n)O(n)O(n)：
